# CacheTrax <small>1</small>

> A hash-indexed object cache with time and size based expiration

- O(1) hash lookups backed by a double linked list
- Automatic size-based pruning on insert
- Time-based expiration with configurable max age
- Custom pruning functions for application-specific eviction
- Works in both Node.js and browser environments
